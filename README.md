# 🐍 Python do Zero ao Avançado | Certificações PCEP, PCAP e PCPP

Curso completo de Programação Python, do nível iniciante ao avançado, com foco em lógica, boas práticas e preparação para certificações internacionais **PCEP™, PCAP™ e PCPP™**.

---

## 📚 O que você vai aprender

### 🔹 Fundamentos da Programação em Python
- Instalação e configuração do ambiente  
- Sintaxe básica e boas práticas  
- Variáveis, tipos de dados e operadores  
- Entrada e saída de dados  

### 🔹 Controle de Fluxo e Estruturas
- Condições (`if`, `elif`, `else`)  
- Laços de repetição (`for`, `while`)  
- Estruturas de dados: listas, tuplas, dicionários e conjuntos  

### 🔹 Funções e Modularização
- Criação e reutilização de funções  
- Parâmetros, retorno e escopo  
- Organização de código em módulos e pacotes  

### 🔹 Programação Orientada a Objetos (POO)
- Classes e objetos  
- Atributos e métodos  
- Encapsulamento, herança e polimorfismo  
- Design de código orientado a objetos  

### 🔹 Manipulação de Arquivos e Erros
- Leitura e escrita de arquivos  
- Tratamento de exceções  
- Boas práticas para código seguro  

### 🔹 Tópicos Avançados de Python
- Compreensões de listas  
- Expressões lambda  
- Iteradores e geradores  
- Manipulação de datas e tempo  

### 🔹 Preparação para Certificações Python
- **PCEP™ – Certified Entry-Level Python Programmer**  
- **PCAP™ – Certified Associate in Python Programming**  
- **PCPP™ – Certified Professional in Python Programming**  
- Exercícios e simulações de prova  

### 🔹 Projetos Práticos
- Aplicações reais com Python  
- Scripts de automação  
- Consolidação do aprendizado com prática
